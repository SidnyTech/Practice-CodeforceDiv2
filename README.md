# Practice-CodeforceDiv2

Este repositorio contiene soluciones a problemas de Codeforces organizadas por fecha de cada concurso.

## Estructura de carpetas

Cada carpeta representa un contest y se nombra usando el formato:

AAAA_MM_DD

- **AAAA** → Año del contest (ej. 2026)
- **MM** → Mes con dos dígitos (01-12)
- **DD** → Día del contest (01-31)

Por ejemplo:

2026_02_02
2026_01_29
2025_12_31

Dentro de cada carpeta, encontrarás las soluciones a los problemas del contest correspondiente.

## Convenciones

- Cada archivo de solución se nombra de forma descriptiva según el problema (por ejemplo, A.cpp, B.py).
- Las carpetas adicionales o notas se pueden incluir dentro de la carpeta del contest.
- Se recomienda mantener siempre el formato AAAA_MM_DD para que la organización se mantenga consistente y clara.

## Objetivo

Este repositorio sirve como **registro personal y estudio** de los contests de Codeforces, permitiendo navegar fácilmente entre concursos por fecha y revisar soluciones anteriores.
